HTML stands for HyperText Markup Language.

Basic HTML Tags

  • <!DOCTYPE html> : Declares the document ty
  • <html> : Root element of an HTML page
  • <title> : Contains meta-information about the document.
  • <body> : Sets the title of the document
  • <h1>, <h2>, ...., <h6> : Headings of different level
  • <p> : Represents a paragraph
  • <a> : Creates a hyperlink
  • <img> : Embeds a image
  • <ul> <ol>,<li> : Creates list
  • <div> : Root element of an HTML page
  • <span> : Defines a section in a document
